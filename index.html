How to connect

The server is a forge server. To connect to it, you have to use the forge launcher and install the same mods as the ones installed on the server. You can find all mods in the same folder as this document.

Install forge: https://files.minecraftforge.net/net/minecraftforge/forge/

(This is darkmode) Follow the installation instructions and make sure you select client

After the instruction, locate the mods folder. On linux it is under $home/.minecraft/mods On windows probably under C:\Programme\minecraft\mods Copy all mods from the gdrive folder into the mod folder Start minecraft
